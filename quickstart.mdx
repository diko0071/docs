### Создание канала для чата
- В боковом меню выберите «Каналы».
- Если существующего канала для чата нет, нажмите «Добавить канал».
- Выберите тип «Чат».
- (Опционально) Измените название канала и настройки виджета, затем сохраните изменения.

### Создание API-канала

- В боковом меню выберите «Каналы».
- Выберите тип «API».
- Заполните название канала.
- В поле Redirect URL укажите URL ниже.
```
https://app.brainfeed.space/api/search/?access_key=<your-api-key>&usedesk=1
```

- Выставите галочку в поле «JSON запрос» и «Отправлять в UTF-8».
- В поле  «Cледить за каналами» выберите название созданного чата из первого пункта.
- Установите отслеживание запросов и чатов на «Новое сообщение».
- Сохраните канал и передайте значение из поля «Секретный ключ» разработке BrainFeed.

## Настройка прав на группы агентов

- Создайте учетную запись для бота или выберите существующую. 

<Warning>Для этой учетной записи должны быть выставлены следующие разрешения: «Работа в каналах», «Просмотр каналов для созданных каналов чата» и API.</Warning>

- Передайте id созданной учетной записи разработке BrainFeed.
- Для группы операторов выставите аналогичные разрешения по работе в каналах и передайте id группы разработке BrainFeed.

## Автоматическое назначение на бота

- Выберите вкладку «Автоматизация» -> «Правила».
- Добавьте новое правило.
- Заполните пункты: 
    1. «Когда»: «Клиент написал сообщение»
    2. «Что»: Все условия «Исполнитель» = «-» 
    3. «Канал» = «Имя созданного чата из начала инструкции»
    4. «Сделать»: «Обновить исполнителя»
    5. «Учетка бота»

## Тестирование виджета

- Во вкладке каналы выберите созданный канал с типом «Чат» из первого пункта.
- В самом низу страницы скопируйте значение из поля «Код виджета».
- Создайте новую базу знаний, в поле скрипты в подвале вставьте ссылку из п.2.
- При переходе к этой базе знаний можно тестировать бота.
